Properties and first-order optimality

First order optimality condition for convex problem in general

For a convex problem
min⁡xf(x)     subject to x∈C \min _{x} f(x) \quad \text { ~~~ subject to } x \in C xmin​f(x)     subject to x∈C
where the feasible set CCC is convex and the convex fff is differentiable, a feasible point x∗x^*x∗ is optimal if and only if
∇f(x∗)T(y−x∗)≥0,∀y∈C \nabla f(x^*)^{T}(y-x^*) \geq 0, \forall y\in C ∇f(x∗)T(y−x∗)≥0,∀y∈C

Proof: If the above first order condition holds, then by the convexity of fff at x∗x^*x∗, we have f(y)≥f(x∗)+∇f(x∗)T(y−x∗)≥f(x∗)f(y)\geq f(x^*)+\nabla f(x^*)^{T}(y-x^*)\geq f(x^*)f(y)≥f(x∗)+∇f(x∗)T(y−x∗)≥f(x∗) So x∗x^*x∗ is optimal.
If x∗x^*x∗ is optimal, i.e., f(y)≥f(x∗),∀y∈C,f(y)\geq f(x^*),\quad \forall y\in C,f(y)≥f(x∗),∀y∈C, to show the first order optimality is true. We assume that ther exists one point z∈Cz\in Cz∈C such that ∇f(x∗)T(z−x∗)<0.\nabla f(x^*)^{T}(z-x^*)<0.∇f(x∗)T(z−x∗)<0. Define yt=(1−t)x∗+tzy_t=(1-t) x^*+tzyt​=(1−t)x∗+tz where t∈[0,1]t\in[0,1]t∈[0,1], we have ddtf(yt)=∇f(yt)(x∗−z)∣t=0<0.\frac{d}{dt}f(y_t)=\nabla f(y_t) (x^*-z)\vert_{t=0}<0.dtd​f(yt​)=∇f(yt​)(x∗−z)∣t=0​<0. So by the mean-value theorem from calculous for very small ttt, we have f(yt)<f(y0)=f(x∗)f(y_t)<f(y_0)=f(x^*)f(yt​)<f(y0​)=f(x∗), which is a controdiction.

In other words, all feasible directions from x∗x^*x∗ (i.e. y−x∗y-x^*y−x∗) are aligned with gradient. When the optimization is unconstrained, this reduces to ∇f(x∗)=0.\nabla f(x^*)=0.∇f(x∗)=0.

While this is a very general condition (and it is sufficient and necessary!), it is difficult to check for most problems and hence is not very useful.
